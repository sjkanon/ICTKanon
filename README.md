# ICTKANON – Website

Een compacte, moderne one‑pager in **HTML/CSS/JS** met secties: Werkplek, Telefonie, Internet, Consultancy, Over en Contact.

## Snel starten
Open `index.html` lokaal in je browser of upload alle bestanden naar je webhosting.

### Aanpassen
- **Logo**: staat in `assets/logo.png` (uit jouw aangeleverde bestand).
- **Kleuren**: pas aan in `styles.css` bij `:root`.
- **Contact**: vervang het mailadres in het formulier (`mailto:info@jouwdomein.nl`).

## Bestanden
- `index.html` – de pagina
- `styles.css` – stijlen
- `app.js` – kleine interacties
- `assets/logo.png` – jouw logo
